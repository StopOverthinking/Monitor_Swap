# Monitor Swap

Monitor Swap은 여러 모니터에 열려 있는 창을 단축키로 한 번에 순환 이동시키는 작은 Windows 트레이 앱입니다.

작업 창을 화면 사이에서 자주 옮겨야 할 때, 창을 하나씩 드래그하지 않고 전체 배치를 왼쪽이나 오른쪽으로 빠르게 넘길 수 있습니다.

## 하는 일

- 선택한 모니터들 사이에서 창을 순환 이동
- 시스템 트레이에서 실행
- 전역 단축키 지원
- 순환에 포함할 모니터 선택 가능
- 최소화된 창 포함 가능
- 영어/한국어 지원

## 빠른 시작

1. `MonitorSwap.exe`를 실행합니다.
2. Windows 시스템 트레이에서 Monitor Swap 아이콘을 찾습니다.
3. 트레이 아이콘을 더블클릭해 **Settings**를 엽니다.
4. 사용할 모니터를 선택합니다.
5. 설정을 저장합니다.
6. 단축키를 눌러 창을 왼쪽 또는 오른쪽으로 순환 이동합니다.

기본 단축키:

- 왼쪽으로 이동: `Ctrl + Win + Shift + Left`
- 오른쪽으로 이동: `Ctrl + Win + Shift + Right`

## 트레이 메뉴

트레이 아이콘을 우클릭하면 다음 메뉴를 사용할 수 있습니다.

- `Rotate Left`
- `Rotate Right`
- `Settings`
- `Start with Windows`
- `Exit`

## 설정

다음 항목을 바꿀 수 있습니다.

- 언어
- 왼쪽/오른쪽 단축키
- 최소화된 창 포함 여부
- Windows 시작 시 자동 실행 여부
- 순환에 포함할 모니터

## 참고

- 최소 두 개 이상의 모니터를 선택해야 제대로 활용할 수 있습니다.
- 이 앱은 트레이에서 실행되므로 메인 창이 계속 열려 있지 않습니다.
- 일부 특수한 시스템 창은 이동하지 않을 수 있습니다.

## 문제 해결

- 아무 일도 일어나지 않으면 모니터가 두 개 이상 선택되어 있는지 확인하세요.
- 단축키가 동작하지 않으면 다른 앱이 같은 단축키를 사용 중일 수 있습니다.
- 앱이 보이지 않으면 숨겨진 트레이 아이콘 영역을 확인하세요.
