# Monitor Swap Docs

이 디렉터리는 Monitor Swap 작업자가 먼저 확인할 내부 문서 공간입니다.

## 문서 구조

- [개발로그](devlog/README.md): 변경 작업, 조사 결과, 의사결정을 시간순으로 기록합니다.
- [위키](wiki/README.md): 프로젝트 구조, 빌드 절차, 반복 참고 지식을 정리합니다.

## 작업 흐름

1. 작업을 시작하기 전에 위키 인덱스와 개발로그 인덱스를 확인합니다.
2. 관련 문서가 있으면 세부 문서를 먼저 읽고 현재 코드와 맞는지 확인합니다.
3. 작업이 끝나면 검증 결과와 남은 후속 작업을 개발로그에 남깁니다.
4. 오래 유지되어야 하는 지식은 개발로그에서 위키로 옮겨 정리합니다.
