# Monitor Swap

[한국어](README_kor.md)

Monitor Swap is a small Windows tray app that moves your open windows across multiple monitors with a hotkey.

If you often rearrange your workspace between screens, this app lets you rotate the whole layout left or right instead of dragging windows one by one.

## What It Does

- rotates windows across selected monitors
- works from the system tray
- supports global hotkeys
- lets you choose which monitors are included
- can include minimized windows
- supports English and Korean

## Quick Start

1. Run `MonitorSwap.exe`.
2. Find the Monitor Swap icon in the Windows system tray.
3. Double-click the tray icon to open **Settings**.
4. Select the monitors you want to include.
5. Save your settings.
6. Use the hotkey to rotate windows left or right.

Default hotkeys:

- Rotate left: `Ctrl + Win + Shift + Left`
- Rotate right: `Ctrl + Win + Shift + Right`

## Tray Menu

Right-click the tray icon to use:

- `Rotate Left`
- `Rotate Right`
- `Settings`
- `Start with Windows`
- `Exit`

## Settings

You can change:

- language
- left and right hotkeys
- whether minimized windows are included
- whether the app starts with Windows
- which monitors take part in rotation

## Notes

- The app works best when at least two monitors are selected.
- It runs in the tray, so no main window stays open.
- Some special system windows may not move.

## Troubleshooting

- If nothing happens, make sure at least two monitors are selected.
- If the hotkey does not work, another app may already be using it.
- If you cannot see the app, check the hidden tray icons area.
